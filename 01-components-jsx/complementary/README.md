# Дополнительный пример: каталог фильмов

Та же идея, что в основном упражнении (список, карточка, загрузка/пустое состояние), но без деления на категории — чтобы увидеть паттерн в чистом виде, без деталей Stellar Burger. Уже решено, TODO нет — читайте, запускайте, меняйте что хочется.

```sh
npm install   # один раз, из корня репозитория (npm workspaces)
npm run dev    # http://localhost:5173
```

Посмотрите `src/components/MovieCatalog.tsx` рядом с `../exercise/src/components/IngredientCatalog.tsx` — структура решения одна и та же.
