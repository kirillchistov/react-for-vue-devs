# Дополнительный пример: блог на React Router

Обычный роутинг — три страницы, вложенный параметр `:slug`, `NavLink` для активного пункта меню — без модальных маршрутов и трюка с `background`. Если основное упражнение показалось перегруженным, начните с этого примера: сама механика React Router тут видна в чистом виде. Уже решено, TODO нет.

```sh
npm install   # один раз, из корня репозитория (npm workspaces)
npm run dev    # http://localhost:5173
```
