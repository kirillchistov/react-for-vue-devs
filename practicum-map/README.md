# Приложение. Как это соотносится с официальной программой

Если вы одновременно проходите официальную программу «React» (Яндекс Практикум) — эта страница помогает сориентироваться: какой модуль курса соответствует какой теме программы, и где искать то же самое другими словами.

Это не конспект и не пересказ материалов программы — только структурная карта соответствий, без единой строчки их текста. Формулировки, примеры и весь код на этом сайте — оригинальные.

## Про дипломный проект `react-burger`

Настоящий дипломный проект — это тот же домен (космическая бургерная), что и сквозной пример этого курса, только собранный по официальному техническому заданию: React + официальная библиотека UI-компонентов Практикума, drag-n-drop сборка бургера, WebSocket-лента заказов, авторизация с refresh-токенами. Ссылка на живую демку — в [README курса](../README.md) и в [10-final-project](../10-final-project).

Если вы уже начали настоящий дипломный проект — самое полезное, что даёт этот курс, это именно те React-специфичные привычки (модальные маршруты, дискриминированные состояния загрузки, `memo`/`useCallback`), которые не зависят от конкретного технического задания и пригодятся в любой версии этого домена.

## Про мини-практики в конце уроков программы

В каждом уроке официальной программы есть небольшая практическая задача — маленький, самодостаточный кусок кода с одной конкретной идеей. Это тот же принцип, на котором построен весь этот курс: `exercise/` → `solution/` → `complementary/` в каждом модуле. Если вам не хватает практики после мини-задачи из урока — тот же модуль курса, но `complementary/`, обычно даёт ещё один самостоятельный пример на ту же идею, в другом домене.
