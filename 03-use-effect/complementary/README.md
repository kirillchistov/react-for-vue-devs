# Дополнительный пример: часы с cleanup

Изолирует cleanup-часть useEffect от сложности с fetch: просто `setInterval` + `clearInterval`. Уже решено, TODO нет.

```sh
npm install   # один раз, из корня репозитория (npm workspaces)
npm run dev    # http://localhost:5173
```

Откройте консоль браузера, нажмите «Спрятать часы» — увидите лог про очистку интервала. Нажмите «Показать часы» снова — новая подписка. Без `clearInterval` в функции очистки старый интервал продолжил бы тикать в фоне даже после того, как компонент исчез с экрана — тихая утечка, которую легко не заметить.
